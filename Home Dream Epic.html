What is Dream Epic?

Dream Epic is an online web application which will help students multitask and efficiently save time to achieve success.

Dream Epic is an all-in-one platform that conveniently contains everything a student needs in one place with a user-friendly interface. There are many different applications available not only for students but also for parents to track their childâ€™s progress and also for teachers to help students to pass classes.

What does Dream Epic offer?

Some helpful features of Dream Epic include an interactive schedule that can help organize activities, a grade system, a system for managing clubs, a system for managing sports, and tutoring service. To add on, our teacher recommendation is one of the most important guiding features in the system.

Our teacher recommendation system builds off the fact that students may be confused on what they really need to focus on.

Teacher recommendation is when a teacher can suggest a user to change the schedule if they are not doing too well in any of the classes and can also add any practice worksheets from the web which is on par with school.
